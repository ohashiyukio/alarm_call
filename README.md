## アプリケーション名 
声掛けアラーム
## アプリケーション概要
時間が来たら入力したコメントを音声にするアプリ

# URL（GitHub）

## テスト用アカウント
ユーザー名：admin
PW:1234

## 利用方法　
トップ画面の「新しいアラームを追加」ボタンから
日時、コメントを入力し「保存ボタン」を押すとセットされる。
時間が来るとコメントを音声出力

## アプリ作成背景
音だけだと何をする時間かわからないので教えてくれるアプリを作ってみました。
